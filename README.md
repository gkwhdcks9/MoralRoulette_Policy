# Moral Roulette 개인정보처리방침

본 앱(모럴룰렛, com.alwayscold.moralroulette)은 사용자의 개인정보를 보호하기 위해 최선을 다합니다.

## 1. 수집하는 개인정보 항목
- 본 앱은 사용자의 개인정보(이름, 전화번호, 이메일 등)를 직접적으로 수집하지 않습니다.
- 게임 내 닉네임, 채팅 등은 서버에 일시적으로 저장되며, 외부로 유출되지 않습니다.
- 앱 사용 중 기기 정보, 위치 정보, 연락처 등 민감한 정보는 수집하지 않습니다.

## 2. 개인정보의 이용 목적
- 게임 서비스 제공 및 원활한 운영
- 서버 상태 모니터링 및 서비스 품질 개선

## 3. 개인정보의 보관 및 파기
- 서버에 저장되는 정보(닉네임, 채팅 등)는 게임 종료 후 일정 기간 내 자동 삭제됩니다.
- 별도의 회원가입, 로그인, 외부 계정 연동은 제공하지 않습니다.

## 4. 제3자 제공 및 위탁
- 본 앱은 개인정보를 제3자에게 제공하거나 외부에 위탁하지 않습니다.

## 5. 접근권한 안내
- 본 앱은 필수 접근권한을 요구하지 않습니다.
- 단, 게임 서비스 제공을 위해 인터넷 연결 권한만 사용합니다.
